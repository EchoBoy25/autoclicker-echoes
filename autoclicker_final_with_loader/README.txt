
GAMER'S AUTO-CLICKER SUITE - INSTALL GUIDE

📦 CONTENTS:
- AutoClicker_Windows.py → Windows version (requires Python + pyautogui, keyboard)
- init.lua → macOS version (for Hammerspoon)

💻 WINDOWS INSTRUCTIONS:
1. Download and install Python from https://python.org
2. Open Terminal or Command Prompt and run:
   pip install pyautogui keyboard
3. Run AutoClicker_Windows.py (double-click or terminal: python AutoClicker_Windows.py)
4. Hotkeys:
   - F8 → Toggle clicker
   - F9 → Toggle key spam
   - F12 → Panic stop

🍎 MAC INSTRUCTIONS:
1. Download and install Hammerspoon from https://www.hammerspoon.org
2. Place init.lua in ~/.hammerspoon/
3. Launch Hammerspoon
4. Allow accessibility permissions when prompted
5. Hotkeys:
   - Option + C → Toggle clicker
   - Option + K → Toggle key spam
   - Option + Shift + P → Panic stop
6. Use the 🎮 menu bar icon to control features via GUI
